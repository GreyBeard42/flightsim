# flightsim
A simple 2D flight simulator made in p5.js

[Play on Homepage](https://greybeard42.neocities.org/javascript/flight_sim/)

Created on March 28, 2024

# Features

- p5 Oscillator

- Browser controls (WASDG)

- Scrolling Background

- 3 plane angles
